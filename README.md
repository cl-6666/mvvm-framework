# Jetpack MVVM框架
一个灵活、高效的Android Jetpack MVVM开发框架，基于AndroidX开发，傻瓜式使用，适用于所有项目，强烈建议以依赖的方式导入，这样避免代码同步，有问题或建议？请通过博客、qq群联系我们。
